Introduzione all'HTML e CSS

Cos'è l'HTML?

HTML è il linguaggio utilizzato per creare il contenuto di una pagina web. Usa tag come <p> per paragrafi e <a> per i link.

Cos'è il CSS?

CSS serve per definire lo stile di una pagina web. Puoi cambiare colori, font, spaziature e molto altro. Ad esempio:

p {
  color: blue;
  font-size: 16px;
}

Consigli per iniziare

  • Scrivi codice semplice e chiaro
  • Usa un editor di testo come VS Code
  • Salva i file con estensione .html e .css
  • Apri i file HTML con un browser
